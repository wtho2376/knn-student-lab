# KNN for Beginners

A minimal, hands-on introduction to the **K-Nearest Neighbors (KNN)** algorithm for high school to first-year college students.

## What's here?
- `index.html`: Interactive webpage with Python & R examples
- `python/knn_example.py`: Standalone Python script
- `r/knn_example.R`: Standalone R script

All examples use the classic Iris dataset and `k=3`.

## How to run
### Python
```bash
pip install scikit-learn numpy
python python/knn_example.py
```

### R
```r
install.packages("class")
Rscript r/knn_example.R
```
